# AI Tarot Reader

An object-oriented Python tarot reading application built with
Streamlit and the OpenAI API.

Users can choose a one-card, three-card, or randomly selected
reading. Each card receives an upright or reversed orientation,
and the application generates an overall reflective interpretation.

## Features

- Object-oriented Card, Deck, Reading, and TarotInterpreter classes
- Complete 78-card tarot deck
- Upright and reversed card meanings
- Random card orientations
- One-card and three-card spreads
- AI-generated combined interpretations
- Streamlit web interface

## Technologies

- Python
- Object-oriented programming
- Streamlit
- OpenAI API
- Git and GitHub
